# HTML Resources

Below is a list of HTML tags that are self-closing for reference:

```
<area />
<base />
<br />
<col />
<embed />
<hr />
<img />
<input />
<link />
<meta />
<param />
<source />
<track />
<wbr />
```

* *In HTML5, the `/` is optional for these items
